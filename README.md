# Cal Clone — Служба онлайн-записи

[![Actions Status](https://github.com/glagius/ai-for-developers-project-386/actions/workflows/hexlet-check.yml/badge.svg)](https://github.com/glagius/ai-for-developers-project-386/actions)

## Описание

Учебный проект — упрощённый клон сервиса [Cal.com](https://cal.com) для организации онлайн-записи на встречи. Позволяет пользователям создавать типы событий, настраивать расписание и принимать бронирования через публичные ссылки.

В проекте есть две условные роли: владелец календаря и гости. При этом в проекте нет регистрации и авторизации. Владелец календаря — один заранее заданный профиль. Этот профиль по умолчанию используется в админской части. Гость бронирует слоты без создания аккаунта и без входа в систему.

Владелец календаря может:

 Создавать типы событий. Для каждого типа события задает id, название, описание и длительность в минутах.
 Просматривает страницу предстоящих встреч, где в одном списке показаны бронирования всех типов событий.
 Гость:

 Может посмотреть страницу с видами брони, где доступно название, описание и длительность.
 Выбирает тип события, открывает календарь и выбирает свободный слот в ближайшие 14 дней.
 Создает бронирование на выбранный слот.
 Правило занятости:

 На одно и то же время нельзя создать две записи, даже если это разные типы событий.
 Окно записи по умолчанию:

 Доступные слоты формируются на 14 дней, начиная с текущей даты.
 Гость может записаться только на свободный слот из этого окна.

## Функционал

- **Типы событий** — создание, редактирование, удаление (название, описание, длительность)
- **Калькулятор слотов** — генерация доступных временных интервалов на 14 дней вперёд
- **Бронирование** — атомарное создание записи с защитой от двойного бронирования
- **Дашборд** — просмотр предстоящих бронирований по всем типам событий
- **Публичное API** — REST-эндпоинты для работы с типами событий и бронированиями

## Стек

| Компонент | Технология |
|-----------|------------|
| Язык | Elixir |
| DSL ресурсов | Ash |
| Data layer | Ash Postgres |
| Web-фреймворк | Ash Phoenix (Phoenix + LiveView) |
| REST API | Ash Json Api |
| UI-компоненты | Phoenix LiveView + Tailwind |
| Контракт API | TypeSpec → OpenAPI 3.0 |
| Базы данных | PostgreSQL |
| Часовые пояса | Timex |
| Деплой | Docker (Docker Compose) |

## Запуск

```bash
make start.dev      # development (Phoenix server + PostgreSQL)
make start.prod     # production (Docker Compose)
```
